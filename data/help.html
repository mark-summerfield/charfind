CharFind

CharFind is an application for finding Unicode characters.

Enter a Search term to find matching Unicode characters.

Any number of search terms can be entered, and each term is one of three kinds. For example a single search term, say, arrow, will match all characters that have arrow in their metadata. Compare this with the multiple search term, say, arrow -down -up -north -south left? right? which means: match all characters that have arrow in their metadata but which do not have down, up, north, or south, and which have either left or right (or both).

So, to match a complete search term, every plain word, word, must be in the metadata, but none of any negated words, -word, and at least one of any optional words, word?.

In addition, a search term can be a decimal or hexadecimal Unicode code point. For example, 2022 will match both a bullet • (whose code point is U+2022) and a character ߦ (whose code point is U+07E6 but decimal number is 2022).

Some initial search terms are provided to help with learning and experimentation, see F2 below.

KeyAction
F1Show this Help window
F2Pop up the Recent Searches—and replace the current search with the chosen one
F3Move the keyboard focus to the Search Results Table
Alt+AAdd the character from the highlighted row in the Search Results Table to the Output Editor
Alt+BShow the About box
Alt+HPop up the History of recently added characters—and add the chosen one to the Output Editor
Alt+CCopy the text from the Output Editor to the clipboard
Alt+OShow the Options dialog
Alt+Q or EscQuit the application
Alt+SMove the keyboard focus to the Search box
